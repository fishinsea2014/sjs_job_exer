Deploying tips:
1. Install node.js
2. Install gulp by "npm i -g gulp"
3. Install bower by "npm install -g bower"
3. Run "bower install"
4. Run "gulp" to start the project.
5. View "http://localhost:3000/" in browser.

Directory structure:
1. "src" is source codes.
2. "build" is codes used to debug program.
3. "dist" is compressed codes used to deploy the project.
